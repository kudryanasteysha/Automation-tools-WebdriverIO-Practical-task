# How to run tests?

## Run all tests
1. Open the terminal
2. Write the command line
```
npm run wdio
```

## Run a single test
1. Write the command in your test. For example:
``` JS
it.only ("should add a new paste", async function() {
    
    })
```
2. Open the terminal
3. Write the command line
```
npm run wdio
```
