# How to run tests?

## Run all tests
1. Open the terminal
2. Write the command line
```
npm run wdio
```

## Run single test
1. Specify **it** as **it.only** :
``` JS
it.only ("should add a new paste", async function() {
    
    })
```
2. Open the terminal
3. Write the command line
```
npm run wdio
```
